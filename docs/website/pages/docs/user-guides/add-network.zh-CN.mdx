# 添加 Rooch 网络到 MetaMask

1. 打开 MetaMask，选择左上角的`网络控制`按钮：

![MetaMask Network Entry](/docs/metamask-network-entry-zh.png)

2. 点击下方的`添加网络`：

![](/docs/metamask-add-network-zh.png)

3. 在添加网络的页面下方点击`手动添加网络`：

![](/docs/metamask-manual-add-zh.png)

4. 在手动添加网络的页面，填写`网络名称`、`新的 RPC URL`、`链 ID` 和 `货币符号` 这四个必填信息。

如下添加 Rooch 的 Dev 网络，并保存：

![](/docs/metamask-devnet-zh.png)
