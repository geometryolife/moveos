# Add Rooch network to MetaMask

Open MetaMask and select the `network control` button in the upper left corner:

![MetaMask Network Entry](/docs/metamask-network-entry-en.png)

Click `Add network` below:

![](/docs/metamask-add-network-en.png)

Click `Add a network manually` at the bottom of the page to add a network:

![](/docs/metamask-manual-add-en.png)

On the page for manually adding a network, fill in the four required information: `Network name`, `New RPC URL`, `Chain ID`, and `Currency symbol`.

Add Rooch's Dev network as follows and save:

![](/docs/metamask-devnet-en.png)
