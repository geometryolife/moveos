# Rooch Portal

## 概述

Rooch Portal 是一个综合仪表板，旨在简化 Unisat 等比特币钱包的管理，并为用户提供直观的界面来管理其比特币和 Rooch 账户。该门户提供了一个集中平台，用于跟踪和组织所有资产、铭文和其他相关金融活动。

## 特性

- 钱包连接：轻松连接和管理您的比特币钱包，例如 Unisat。
- 资产管理：在一处查看和管理您的比特币和 Rooch 账户资产。
- 铭文跟踪：跟踪您的所有铭文和相关活动。
- 用户友好的界面：通过直观且用户友好的仪表板浏览您的财务信息。
- 安全可靠：优先考虑财务数据的安全和隐私。

## 使用介绍

这部分主要引导用户熟悉和使用 Rooch Portal。

### 首页

- Rooch Portal 网址: TODO

打开 Rooch Portal 网址后，如下图所示：

![](/docs/portal/portal-home.png)

这是 Rooch Portal 的首页，主要展示 Rooch 账户的资产总览，将会在 Assets 部分详细介绍。

左边的侧边栏是功能导航区：包括 Assets、Mint、Apps、Transactions、Settings。

左下角是界面语言和界面明暗主题切换的按钮，以及当前连接的地址展示。

右上角是连接钱包的按钮。

### 连接钱包

点击右上角的 `Connect Wallet` 按钮，将提示连接钱包，点击连接 Unisat Wallet，如果还没有安装，请参阅[安装 UniSat](./install-unisat.zh-CN.mdx)。

![](/docs/portal/connect-wallet.png)

连接成功之后：

![](/docs/portal/assets.png)

注意左下角和右上角，右上角连接的是 UniSat 的比特币账户地址，右下角显示的是 Rooch 账户地址，并且这两个地址已经相互关联。

### Assets

这是 Rooch Portal 首页的默认功能区。

![](/docs/portal/assets-show-coin.png)

右上角显示当前账户的余额、Rooch 账户地址、关联的公链主网地址（BTC、ETH 等）。

左边的四个按钮是展示不同种类资产的导航按钮，默认展示 Coin，即同质化资产。

下面不同资产的展示区，右下角是翻页按钮。

#### Coin

同质化资产是最常见的各个链的治理代币，比如 BTC、ETH、SOL 等。

如上图所示，Coin 默认展示 5 个代币，可点击 `Previous/Next` 进行翻页或直接点击显示的页码。

#### NFT

NFT 展示的是我们常见的非同质化资产。

![](/docs/portal/assets-nft.png)

#### SFT

SFT 展示的是半同质化资产。

![](/docs/portal/assets-sft.png)

#### UTXO

Indexed Assets 展示的是一些索引资产，比如 UTXO。

![](/docs/portal/index-assets.png)
