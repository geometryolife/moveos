Rooch Network 旨在实现两个强大且快速发展的社区：Move 和比特币之间的无缝集成。在 Rooch 创新技术堆栈的支持下，开发人员首次可以使用先进的 Move 语言在比特币上进行构建，而用户则可以享受安全且轻松的比特币 DApp。

作为一种无需许可的去中心化协议，Rooch Network 依靠基于加密代币的经济系统来维持其功能和运营，并加强其社区和生态系统。

在这篇文章中，我们很高兴介绍 Rooch Network 的代币经济学和官方 $ROOCH 代币。

## $ROOCH代币

以下是$ROOCH代币的基本信息：

- 代码： ROOCH
- 总供应量：10,000,000,000
- 十进制：12
- 代币格式：Rooch Native Coin

$ROOCH 代币在 Rooch 生态系统中发挥着重要作用。其预定义功能包括：

价值存储：$ROOCH 代币与 Rooch 网络的汽油费代币 $RGAS 无缝协作。用户可以存入$ROOCH来借入$RGAS。此外，网络消耗的所有 $RGAS 的 40% 将用于回购和销毁 $ROOCH，从而提升其价值。
治理：$ROOCH 作为 Rooch 生态系统内的主要治理代币，使代币持有者能够参与关键决策过程。
网络安全：Rooch 节点运营商需要 $ROOCH 代币作为押金来加入并帮助保护网络。

Rooch Network aims to enable seamless integration between two strong and rapidly growing communities: Move and Bitcoin. Powered by Rooch’s innovative tech stack, developers can, for the first time, build on Bitcoin using the advanced Move language, while users can enjoy secure and effortless Bitcoin DApps.

## $ROOCH Distribution

![](/blog/rooch-tokenomics/img2.png)

### 社区

- 20% - 2,000,000,000 $ROOCH

Rooch Network将分配代币总供应量的20%来支持社区活动的各个角度。

- 发展比特币活动：

$ROOCH 总供应量的 4.2% 被分配作为该活动的激励措施，旨在加速更广泛的比特币社区的发展。在代币生成事件（TGE）后，这些代币将基于 BitXP 系统分发给用户。 TGE后，将持续开采额外的代币并奖励给用户。

- 链上激励：

$ROOCH 总供应量的 5.8% 用于根据社区用户的链上活动（例如参与邀请计划和与 DApp 互动）奖励社区用户。

- 初始开发者支持：

$ROOCH 供应总量的 5% 被保留用于补偿开发者的链上成本。智能合约开发人员将根据用户与其部署的合约的交互获得汽油费奖励。

- 初始流动性支持：

$ROOCH 供应总量的 5% 用于奖励链上去中心化交易所 (DEX) 上的流动性提供者。

### 生态系统

40% - 4,000,000,000 $ROOCH

Rooch Network 将分配代币供应总量的 40%，以促进 Rooch 生态系统的增长和发展。详细情况如下：

生态系统补助金（20%）：

这种分配支持各种生态系统项目和活动，包括必要的基础设施开发、旗舰应用程序以及线上和线下社区活动。

节点运营商（20%）：

该分配致力于奖励和补偿节点验证者和最终确定性提供者，有助于确保Rooch网络的顺利运行和维护。
40% - 4,000,000,000 $ROOCH

### 团队

40% - 4,000,000,000 $ROOCH

Rooch团队将预留代币总量的40%，以确保Rooch协议和生态系统的可持续发展。分配详情如下：

- 早期支持者 (17%)：

这部分分配给Rooch支持者，并将在36至48个月的时间内逐步释放。

- 核心贡献者（23%）：

该分配是为 Rooch 协议核心团队成员保留的，并将在 48 至 60 个月的时间内释放。

## 发布时间表

![](/blog/rooch-tokenomics/img3.png)

在代币发行方面，所有$ROOCH代币都将由智能合约托管，按月发行。

团队代币在 TGE 后的前 6 个月内锁定，并将以个人速度逐步向早期支持者和核心贡献者解锁。
生态系统代币在 TGE 上解锁了总代币供应量的 3%，其余的在 TGE 后线性解锁并投入使用。
社区代币均通过不同的活动和计划分发给社区成员。它的发布取决于社区的参与。 Rooch Network 团队将设计用于从智能合约中释放代币的初始程序，但随着生态系统的发展，基于去中心化治理的机制最终将接管此功能并做出社区代币释放的决定。

### 代币发行

在代币发行方面，所有$ROOCH代币将由智能合约托管并按月发行。

团队代币：

这些代币将在代币生成事件（TGE）后的前六个月内被锁定，然后将以不同的速度逐步为早期支持者和核心贡献者解锁。

生态系统代币：

在 TGE 上，代币供应总量的 3% 将被解锁，其余部分将线性解锁并随后投入使用。

社区代币：

这些代币将通过各种活动和计划分发给社区成员。发布时间表取决于社区参与。最初，Rooch Network 团队将设计从智能合约中释放代币的程序。然而，随着生态系统的发展，去中心化的治理机制将接管这一功能，决定社区代币的释放。
